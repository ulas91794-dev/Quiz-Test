# Quiz-Test
HTML, CSS ve JavaScript kullanarak geliştirdiğim bu interaktif bilgi yarışması uygulaması, kullanıcıların keyifli vakit geçirirken kendilerini test etmelerini sağlar. Dinamik soru yapısı, anlık skor takibi ve şık arayüz tasarımı ile hem mobil hem de masaüstü cihazlarda kusursuz çalışır. Kod kalitesini ve web performansını artırmayı hedefledim.
